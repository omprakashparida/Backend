# Portfolio Backend API


## 🔧 Local Development

```bash
npm install
npm run dev
```

## 📡 API Endpoints

- `POST /api/contact/submit` - Submit contact form
- `GET /api/contact/stats` - Get contact statistics
- `GET /api/contact/all` - Get all contacts (admin)

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)
- Nodemailer
- CORS
- Helmet (Security)
- Rate Limiting
